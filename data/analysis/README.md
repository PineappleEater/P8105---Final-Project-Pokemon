Comprehensive Pokemon rating and tier systems.

**Files:**
- `pokemon_rated_dual.csv` - Dual rating system (offensive/defensive)
- `pokemon_rated_single.csv` - Single composite rating system
- `rating_summary_*.csv` - Summary statistics
- `top_*_pokemon.csv` - Top Pokemon lists by category
- `README.md` - Detailed documentation

See `rating/README.md` for detailed documentation on the rating systems.

### 3. Type Analysis (`type/`)

**Files:**
- Type effectiveness matrices
